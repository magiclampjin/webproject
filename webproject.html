BIG DATA,

Data Mining

`

DEFINITION

(출처: IT용어사전)

빅데이터를 분석하는 기술로 관찰이나 실험, 조사를 통해서 얻은 사실을 뜻하는 데이터와 광산에서 광석을 캐내는 채굴을 뜻하는 마이닝이 만나 만들어진 단어이며, 광산에서 광석을 캐내는 것처럼 수많은 데이터 속에서 유의미한 정보를 찾아내는 것을 의미한다. 즉, 대규모로 저장된 데이터 안에서 관계나 통계적인 규칙, 패턴을 탐색하고 찾아내는 일련의 과정들로, 데이터의 형태와 범위가 다양해지고 그 규모가 방대해지는 빅데이터의 등장으로 데이터 마이닝의 중요성은 부각되고 있다.

PROCESS

데이터 마이닝의 방법론은 다양하지만 잘 알려진 방법론으로는
크게 CRISP-DM, SEMMA, KDD의 3가지가 있다.
각 방법론은 필요에 따라 변형되어 적용되기도 한다.

CRISP-DM

전 세계에서 가장 많이 사용되는 데이터 마이닝 표준 방법론으로 계층적 프로세스 모델이다. 이 방법론의 절차는 6단계로 이루어져 있으며 각 단계는 순차적으로 진행되는 것이 아니라 필요에 따라 반복 수행되며 분석의 품질을 향상한다.

SEMMA

SAS에서 사용하는 방법론으로 샘플 단계, 탐색 단계, 수정 단계, 모델 단계, 평가 단계의 5단계로 구성된 프로세스 모델로, 각 단계 실행 후 문제가 발생하면 전 단계로 돌아가 피드백이 되는 프로세스 형식을 가지고 있다.

KDD

데이터를 통해 통계적 패턴이나 지식을 찾을 수 있도록 정리한 데이터 마이닝 프로세스로 선택, 전처리, 변환, 데이터 마이닝, 결과 평가의 5단계로 이루어져 진행된다. 만약 결과가 충족되지 않았다면, 데이터 선택과 데이터 마이닝 절차를 반복 수행하는 방식으로 진행한다.

ANALYTICAL TECHNIQUE

데이터 방법론이 다양한 것처럼 데이터 마이닝을 사용하는 분석 기법들의 종류도 다양하다. 데이터 특성화, 차별화를 통해 목표 변수를 설명 변수의 함수모델로 생성하는 작업인 예측 모델링, 데이터에 강하게 연관되는 특징을 설명하는 패턴을 발견하는 연관 분석, 같은 군집에 속하는 관측을 다른 군집에 속하는 관측보다 더 유사하도록 긴밀하게 관련된 관측 그룹을 탐색하는 군집분석 등 다양한 범위의 데이터 분석과 지식 발견 작업을 한다.

REFERENCE

UESCASE

데이터 마이닝은 다양한 분야에서 활용되고 있다. 많은 활용 사례가 있지만 가장 대표적인 사례들은 다음과 같다. 먼저, 미국에서 맥주와 기저귀의 판매량이 서로 강한 연관관계를 가지고 있다는 것을 알아내 이를 함께 진열함으로써 수익을 높인 사례가 있고, 국가정보기관에서 엄청난 양으로 도청되는 통신들 사이에서 가장 중요한 통신을 찾아내기 위해서 전문가들이 네트워크를 위협하는 문제가 있는지를 알아내는 과정에서도 사용된 사례가 있으며, 인터넷 쇼핑몰의 상품 추천 시스템에서 고객들의 소비패턴이나 성향을 분석하여 적합한 상품을 추천하는 기능을 만든 사례가 있다. 데이터 마이닝은 이외에도 금융, 교육, 경영 등 다양한 분야에서 사용되고 있다.
